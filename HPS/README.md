## Квадрат Гарри Поттера

Использование:

В терминале запустить php скрипт, например:

`php hps/runner.php 01` - где 01 номер заклинания

Заклинания, с учетом того, что размер поля 25х25, а первая точка [0;0]:

1. `$y > $x`
2. `$y === $x` 
3. `$y === 24 - $x`
4. `$y <== 29 - $x`
5. `$y === 2 * $x + $y % 2`
6. `$x <= 9 || $y <= 9`
7. `$x >= 16 && $y >= 16`
8. `$x * $y == 0`
9. `$y < $x - 10 || $y > $x + 10`
10. `$x < $y && $y < 2 * $x + 2`
11. `$x == 1 || $y == 1 || $x == 23 || $y == 23`
12. `25 >= ($y * $y + $x * $x) / 16`
13. `$y > 20 - $x && $y < 28 - $x`
14. `$y <= 100/($x + 1)`
15. `$y > 20 - $x && $y < 28 - $x`
16. `$y > 20 - $x && $y < 28 - $x`
17. `$y > 20 - $x && $y < 28 - $x`

20. `$x % 2 == $y % 2`
